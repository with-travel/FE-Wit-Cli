# 이전 변경사항

## [1.0.58] - 2025-09-07

**신규 기능**
- 입력 필드에 선택적 중복확인 버튼 추가(기본 라벨 ‘중복확인’) — 입력과 버튼이 가로 배치되고 onCheck로 검사 트리거.

**리팩토링**
- 이메일 가입 화면을 단일 입력으로 통합해 중복확인·유효성·오류/안내를 일관 처리.
- 여행 성향 문항을 문항별 컨트롤러로 분리해 개별 검증 적용.
- 추가정보의 생년월일을 YYYY-MM-DD 문자열로 저장하고 한국어 형식으로 표시.
- 일부 내부 API 경로 및 빌드 스크립트 구조 정리, iOS 빌드 항목 정리.

**문서**
- CHANGELOG 구조 및 포맷 재정비, README에 최신 릴리스 반영.

**작업 정리**
- 앱·패키지 버전 1.0.57 → 1.0.58으로 업데이트.
